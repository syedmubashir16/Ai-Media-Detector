01 — Overview

Abstract

A summary of the project, methodology, and key findings.

The rapid proliferation of generative AI tools such as DALL-E, Stable Diffusion, Midjourney, Runway ML, and Pika has made it increasingly difficult to distinguish between human-created and AI-generated media. This project presents AuthentiScan — a machine learning system for detecting AI-generated images and videos using classical feature engineering and scikit-learn classifiers, deployed as a live web application.

The system extracts 96-dimensional feature vectors from images (DCT coefficients, LBP histograms, color statistics, noise residuals, edge features) and 222-dimensional vectors from videos (adding optical flow and temporal consistency features). Four classifiers — Random Forest, SVM (RBF), Gradient Boosting, and XGBoost — were trained and compared on multiple datasets.

The system achieved 92.4% accuracy on image detection and 82.4% accuracy on modern AI-generated video detection. A key finding is that modern AI content (2024 generators) is significantly more detectable than legacy compressed deepfakes (DFDC 2019), with a 20 percentage point accuracy difference. The system is deployed on HuggingFace Spaces with a user feedback loop and anti-poisoning protection.

Feature Engineering

Raw pixels are not used directly. Instead, discriminative statistical and frequency-domain features are extracted.

📊
DCT Coefficients
2D Discrete Cosine Transform applied to 8×8 blocks. Statistical moments computed over 4 frequency bands. Captures AI upsampling artifacts.
16 dimensions
🔲
LBP Histogram
Local Binary Patterns with R=1, P=8. Uniform patterns capture micro-texture. AI images show unnaturally smooth texture distributions.
59 dimensions
🎨
Color Statistics
Mean, standard deviation and skewness computed per RGB channel. AI generators often produce unnatural color distributions.
9 dimensions
📡
Noise Residuals
High-frequency noise extracted by subtracting Gaussian blur. Real photos contain camera sensor noise absent in AI images.
8 dimensions
🔎
Edge Features
Sobel edge map statistics and Canny edge density. AI images often have over-smoothed edges with unnatural density patterns.
4 dimensions
🎬
Temporal Features (Video)
Optical flow magnitudes and inter-frame difference statistics. Captures temporal inconsistencies unique to AI-generated video.
30 dimensions

💡
Key Research Finding
Modern AI-generated content (Runway ML, Pika, Luma AI — 2024) is significantly more detectable by classical ML than legacy compressed deepfakes from DFDC (2019), with a 20 percentage point accuracy difference (82.4% vs 62%). This demonstrates that classical ML features — particularly DCT frequency artifacts and LBP texture patterns — are highly effective against contemporary AI generators but struggle with heavily compressed legacy deepfake datasets where compression destroys the artifact signatures. This suggests that classical ML remains a viable and practical approach for real-world AI media detection in 2026.
05 — Continuous Learning

User Feedback & Anti-Poisoning

A feedback collection system with multiple layers of protection against data poisoning attacks.

Feedback Collection Flow

1
User Submits
User corrects a prediction using the feedback form on the web app
2
Anti-Poisoning Check
Rate limiting, confidence check, and agreement validation
3
Pending Review
Feedback held in pending state awaiting agreement from other users
4
Agreement Verified
3+ users must agree on same label before marking as verified
5
Training Queue
Verified feedback added to training dataset for periodic retraining

Anti-Poisoning Protections

⏱️ Rate Limiting
Maximum 10 feedback submissions per IP address per day. IPs stored as hashed values for privacy.
🛡️ Confidence Guard
Feedback contradicting predictions with >85% confidence is automatically rejected to prevent poisoning.
🤝 Agreement System
Minimum 3 independent users must agree on the same label before any feedback is considered verified.
👁️ Admin Review Panel
All pending and verified feedback visible at /admin panel for manual review before inclusion in training.
06 — Analysis

Limitations & Future Work

⚠️ Current Limitations
❌ Compressed videos (WhatsApp, Telegram) cause false positives — compression removes noise artifacts
❌ 62% accuracy on DFDC legacy dataset — heavy H.264 compression destroys frequency features
❌ Classical ML ceiling ~65% for compressed video detection
❌ News broadcasts with text overlays reduce classification confidence
❌ Small modern video dataset (50 videos per class)
🚀 Future Work
✓ Deep learning (3D CNN, Vision Transformer) for video spatiotemporal features
✓ Add compressed real videos (WhatsApp, Telegram) to training data
✓ Expand modern AI video dataset to 500+ samples per class
✓ Browser extension for real-time web content verification
✓ Audio deepfake detection integration
✓ Federated learning for privacy-preserving model updates
07 — Summary

Conclusion

A summary of achievements, contributions and research findings.

This project successfully developed and deployed a classical machine learning system for detecting AI-generated images and videos. The system achieves 92.4% accuracy on image detection and 82.4% accuracy on modern AI-generated video detection, demonstrating that classical ML with engineered features is highly effective against contemporary AI generators including Runway ML, Pika, and Luma AI.

Unlike deep learning approaches that require GPUs and millions of parameters, this system operates entirely on CPU with sub-3 second inference times, making it deployable without specialized hardware. The FastAPI backend and web frontend provide a complete end-to-end solution accessible to non-technical users, deployed live on HuggingFace Spaces.

The key research finding is that modern AI-generated content is significantly more detectable than legacy compressed deepfakes, with a 20 percentage point accuracy difference. This suggests that the AI detection problem may actually become easier as AI generators improve, since better generators introduce more consistent and detectable artifact patterns in frequency and texture domains.

The integrated feedback system with anti-poisoning protection provides a foundation for continuous model improvement, ensuring the system can adapt to new AI generation techniques as they emerge. This work contributes a practical, interpretable, and deployable tool to the growing field of AI media forensics.
