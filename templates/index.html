How it works

Classical ML pipeline with engineered features — no GPU required.
01
Feature Extraction
DCT coefficients, LBP texture histograms, color stats, and noise residuals are extracted from each frame.
02
Video Aggregation
For videos, frames are sampled at 1 FPS and features are aggregated with mean + std pooling and temporal diff stats.
03
Classification
A Random Forest classifier trained on CIFAKE & FaceForensics++ scores the 96–204 dimensional feature vector.
04
Explainability
Feature importance from the ensemble model shows which signals drove the prediction.
Built with Python · scikit-learn · Flask · Final Year Project 2024/25
